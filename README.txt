O objetivo deste trabalho é realizar a construção de um aplicativo STREAMLIT utilizando as funções básicas da biblioteca, 
como write, markdowns, inclusão de tabelas e arquivos de mídia;

Para correto funcionamento, o Streamlit deve ser inicializado dentro da pasta "App" executando o comando abaixo via terminal powershell.
Streamlit run St_Spotify_Home.py --server.port=8087 --browser.serverAddress='177.192.12.238';

Devem ser instalados todos os pacotes listados no Pipfile;

Para melhorar a diagramação, foram combinados comandos CSS, amplamente divulgados na comunidade como solução alternativa,
visto que as funções do Streamlit carecem de parâmetros para customização visual;
